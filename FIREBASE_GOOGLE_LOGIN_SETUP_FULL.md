# Firebase Google Login Setup Guide

This guide will help you set up Firebase Google Login for your Aarambh LMS application.

## Prerequisites

1. You already have a Firebase project (aarambh-01) with service account credentials
2. You need to set up a Web app in Firebase Console to get the Web API Key

## Step 1: Get Firebase Web Configuration

1. Go to [Firebase Console](https://console.firebase.google.com/)
2. Select your project "aarambh-01"
3. In the left sidebar, click the gear icon (Project Settings)
4. In the "General" tab, scroll down to "Your apps"
