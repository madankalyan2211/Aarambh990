# Teacher Video Upload Guide

## Overview
This guide explains how teachers can upload video content directly to their course lessons. Students will be able to view these videos when accessing the lesson content.

## Prerequisites
- Teacher account with course creation/management permissions
- Video files in supported formats (MP4, WebM, etc.)
- Video files under 100MB in size

## Step-by-Step Instructions

### 1. Access Course Management
1. Log in to your teacher account
2. Navigate to "My Courses" 
3. Click on the course you want to add video content to
4. Click "Manage Content" button

### 2. Create or Edit a Lesson
1. If creating a new lesson:
   - Click "Add Lesson" in the appropriate module
2. If editing an existing lesson:
   - Click the edit icon (pencil) next to the lesson

### 3. Set Lesson Type to Video
1. In the lesson form, find the "Lesson Type" dropdown
2. Select "Video" from the options

### 4. Upload Video File
1. Scroll down to the "Upload Video File" section
2. Click "Choose Video File" button
3. Select your video file from your computer
4. Wait for upload to complete (progress bar will show status)
5. You'll see a confirmation when upload is successful

### 5. Alternative: Use External Video URL
Instead of uploading, you can also use:
- YouTube video URLs
- Vimeo video URLs
- Enter the URL in the "Video URL" field

### 6. Add Additional Content (Optional)
1. Add a title for your lesson
2. Include written content in the "Content" text area
3. Set duration and order as needed

### 7. Save Lesson
1. Click the "Save" button
2. Your lesson with video content is now available to students

## For Students

### Viewing Uploaded Videos
1. Students log in to their accounts
2. Navigate to "My Courses"
3. Select the course containing the video lesson
4. Click on the lesson with the video content
5. The video will play directly in the browser

## Supported Video Formats
- MP4 (H.264 codec recommended)
- WebM
- Other browser-compatible formats

## File Size Limits
- Maximum file size: 100MB
- Recommended: Keep videos under 50MB for best performance

## Best Practices
1. **Compress videos** before uploading to reduce file size
2. **Use descriptive titles** for lessons to help students find content
3. **Include transcripts** in the content area for accessibility
4. **Test playback** after uploading to ensure videos work correctly
5. **Provide alternative formats** when possible (e.g., YouTube link as backup)

## Troubleshooting

### Upload Issues
- **File too large**: Compress your video or split into smaller segments
- **Unsupported format**: Convert to MP4 or WebM format
- **Upload stuck**: Refresh the page and try again

### Playback Issues
- **Video won't play**: Check browser compatibility
- **Poor quality**: Upload a higher quality version
- **Long loading times**: Video may still be processing

## Technical Details

### Storage
- Videos are stored securely on our servers
- Files are given unique names to prevent conflicts
- Storage space is monitored to ensure availability

### Security
- Only authorized teachers can upload videos
- All uploads are scanned for security
- Access is restricted to enrolled students only

### Performance
- Videos are optimized for web playback
- Content delivery network (CDN) ensures fast loading
- Adaptive streaming adjusts quality based on connection speed
